O que é Linux Mint?

É uma distribuição Linux Irlandesa que foi concebida por Clément Lefèbvre e que é ativamente desenvolvida pela equipe Linux Mint Team e por sua comunidade. Baseado em Ubuntu (Distribuição baseada em Debian) e Debian (Baseado em Linux kernel). O Mint tem 3 tipos de ambientes: Cinnamon (Versão mais popular e mais atrativa visualmente), MATE (Versão mais estável que a Cinnamon) e Xfce (Tem interface mais simples, indicado para computadores mais simples e de baixo desempenho).

Objetivo do site

O Windows é atualmente o sistema operacional mais popular, mas seu custo pode chegar a aproximadamente 500 reais, o que é considerado absurdo para a maioria das pessoas. Em contraste, o Linux surge como uma alternativa gratuita e, além disso, muito mais segura para os usuários, uma vez que seu código é aberto.

O Linux foi concebido desde o início como um projeto sem fins lucrativos. Sua comunidade é muito engajada trabalhando bastante para criar novas distribuições, melhorar a segurança e a interface do sistema, tornando-o amplamente utilizado em todo o mundo. Embora seja um pouco mais complexo do que o Windows, não é algo impossível de se aprender, no entanto, muitas pessoas ainda relutam em fazer a transição para o Linux por esse motivo.

Este site tem como objetivo principal mostrar que fazer a troca de seu sistema operacional (incluindo ensinar o boot) não é uma tarefa difícil, e que existem distribuições Linux excelentes, leves e seguras disponíveis para todos os tipos de usuários.

Optamos por destacar o Mint como uma opção recomendada, especialmente para novos usuários que nunca tiveram contato com o Linux. O Mint é reconhecido por sua qualidade desde o seu lançamento em 2005, tornando-se uma ótima escolha como sistema operacional.
