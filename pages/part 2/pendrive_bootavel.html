Como fazer um pendrive bootavel?

Passos Iniciais do Linux Mint Vantagens e desvantagens do Mint Teste seu conhecimento

Vamos utilizar o programa “Ventoy”, que permite transformar o pendrive numa unidade com boot, suportando arquivos ISO. Para baixar o Ventoy primeiro vamos entrar no site oficial Site oficial Ventoy

Após entrar no site vamos baixar o programa clicando na opção: ventoy-1.0.97-wind

Após clicar, o usuário será direcionado para o site SourceForge, site que os desenvolvedores utilizam para guardar os programas, temos que clicar no ventoy-10.0.97-windows.zip

Após clicar nele, o usuário vai ser redirecionado para o download automático. Após o tempo acabar irá mostrar a tela do explorador de arquivos para o usuário escolher onde vai salvar

Após escolher o local e clicar em salvar. Apenas devemos clicar com o botão direito e extrair o arquivo. Lembrando que: Se tiver um software compactador e descompactador de dados (Exemplo: Winrar e 7-zip) tem como extrair da mesma maneira. No tutorial está sendo usado o nativo do Windows.

Após clicar em “Extrair Tudo…” devemos escolher onde queremos extrair, nesse caso, vamos extrair tudo na mesma pasta. Após clicar em “Extrair”, começará a carregar e vai aparecer a mesma pasta, porém, extraída.

Já na pasta vamos ter todos os arquivos do programa, para instalá-lo precisamos abrir o “Ventoy2DIsk”, pois, se trata do programa. Quando você clicar no arquivo, o windows perguntará se você deseja mesmo abrir, para fins de segurança isso é um procedimento padrão, devemos clicar em SIM pois o ventoy é código aberto e não apresenta malware.

Após ter clicado em abrir, irá mostrar a interface do software. Com um compartimento chamado “device” que significa o pendrive que está espetado no PC. Detalhe: é importante abrir o programa com o pendrive espetado. Caso espete o pendrive depois, clique no ícone verde ao lado do device. Após aparecer seu pendrive clique em “install” e ele fará a dupla verificação se você realmente deseja formatar o pendrive, pois, apagará todos os dados contidos no pendrive.

Após clicar ‘’Sim’’ em todos os avisos, irá começar a carregar. Quando finalizar, a mensagem ao lado irá aparecer.
Essa mensagem informa que o Ventoy foi instalado com sucesso no pendrive!
Detalhe: ele trocará o nome do pendrive por “Ventoy”.

Você deve selecionar a imagem baixada com Ctrl + X (recortar o arquivo que está salvo no computador para o pendrive) e clicar no pendrive e colar com o Ctrl + V. Após carregar, pronto, a ISO está no pendrive pronta para dar o boot no computador. Lembrando que é possível colocar várias ISO’s no mesmo pendrive e escolher qual dar o boot. Essa é a tela do ventoy quando vai dar boot no computador, vemos várias distribuições de Linux e algumas ISOs do Windows, com a seta do teclado é possível escolher qual quer dar boot e clicando no “enter” será escolhido.

Dando boot no computador

Com o pendrive pronto e espetado no gabinete, vamos dar o boot no computador!
Boot é iniciar o computador, apenas temos um boot bem sucedido quando temos o SO (sistema operacional) completo, nesse caso estamos utilizando o termo para formatar o computador num novo SO. Siga o passo a passo:

Desligue o computador;

Ligue novamente o computador;

Assim que ligar, aperte o botão “delete” do seu teclado para conseguir entrar na BIOS do seu computador. O botão de iniciar BIOS (Conjunto de instruções que permite que o processador trabalhe junto com os periféricos) pode variar de acordo com cada computador, normalmente é o “delete” do teclado. Caso não funcione, tente as teclas: Esc, F1, F2, F10 e F12 (não em sequência, tente uma de cada vez);

Na tela de BIOS, vá na parte de Boot no canto superior;

Quando estiver na aba de Boot coloque o seu pendrive como o “boot 1”. Ele deve ser o primeiro a iniciar junto com o windows;

Após selecionar o pendrive como primeiro, vá na última aba que tem “Save and exit” e clique na opção “Save and exit”;

Seu computador irá reiniciar e vai abrir a tela inicial do Ventoy e assim você clica com “enter” e vai começar a formatação!
