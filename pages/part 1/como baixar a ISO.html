Como baixar a ISO

Primeiramente precisamos decidir qual dos ambientes de trabalho vamos escolher. Após decidir, vamos abrir o site original do Mint Link Site Oficial Linux Mint Download da ISO. Vamos basear nosso tutorial pelo Cinnamon, porém, mesmo que o usuário escolha outro ambiente, a forma de instalação segue sendo a mesma. Vamos clicar no botão de “Download” como mostra na imagem ao lado.

Após isso, temos a opção de baixar a ISO por Torrent, caso o usuário tenha, é possível clicando no Torrent Download: 64-bit, como mostra na imagem, onde aponta a seta verde: Após isso, o usuário deve clicar no arquivo e vai ser direcionado para o Torrent. Caso o usuário não tenha, é possível baixar um espelho da distribuição, que está logo abaixo da página.

É importante escolher aquele que for mais próximo de sua localização atual. Nesse caso vamos escolher o do Brasil.

Clicando em alguma delas vamos ser direcionados para o download da distribuição.

Após escolher o local onde vamos salvar, a imagem vai ser baixada e a partir daí teremos ela salva em nosso computador. A imagem pode ser utilizada para tanto formatar o computador num pendrive bootável (que trocamos o sistema operacional) ou numa máquina virtual (que usamos um software para simular a imagem).
